# How to install Apache Http server on Ubuntu
* Search Apache package with “apt-cache search” command
	* apt-cache search apache

* Get Apache package name “apache2”, issue the “apt-get” install command. You need “sudo” privileged for this installation.
	* sudo apt-get install apache2

* Done, run your favor internet browser, e.g firefox or IE. Type “http://localhost” in your browser’s URL text box.
